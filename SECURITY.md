# Security

## Supported Versions

Security fixes are expected to target the latest version.

## Reporting A Vulnerability

Please open a private report or contact the maintainer directly if the repository has private vulnerability reporting enabled.

Avoid posting exploit details publicly until there has been time to investigate.

## Scope

Security-sensitive areas include:

- Extension permissions.
- Host permissions.
- Caption extraction from third-party pages.
- Translation request handling.
- Local cache handling.
