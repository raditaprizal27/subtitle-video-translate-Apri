# Changelog

## 3.1.0

- Rebranded extension as Apri Video Translate.
- Replaced AI-provider configuration with a simple Google Translate based flow.
- Added Indonesian as the default target language.
- Added bilingual subtitle overlay with translation first and original caption second.
- Added translation cache in local extension storage.
- Added debounce around subtitle detection.
- Added YouTube, YouTube embed, Udemy, and generic HTML5 video support.
- Simplified popup and options UI.
- Reduced permissions and kept Manifest V3 compatibility.
